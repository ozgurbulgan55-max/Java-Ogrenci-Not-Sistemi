# Java-Ogrenci-Not-Sistemi
Konsol tabanlı geliştirilen Java uygulaması; öğrencilerin vize–final notlarını alarak ortalama hesaplayan, harf notu belirleyen ve sınıf ortalaması çıkaran modüllerden oluşmaktadır. Projede nesne yönelimli programlama prensipleri (fonksiyon yapıları, metodlar), kullanıcıdan veri alma, koşullu ifadeler, döngüler ve hata kontrolü uygulanmıştır.
